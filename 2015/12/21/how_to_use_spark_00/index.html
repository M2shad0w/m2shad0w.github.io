数据平台spark

简介

Apache Spark 是一个快速、通用的计算机集群系统。它提供高度优化 Java, Scala, Python and R 的高级接口来支持一般的执行流程，同时支持丰富的高层及的工具集合，包括 Spark SQL来处理关系型数据，MLlib来处理机器学习，GraphX来处理图运算，Spark Streaming

系统要求

Spark runs on Java 7+, Python 2.6+ and R 3.1+. For the Scala API, Spark 1.5.2 uses Scala 2.10. You will need to use a compatible Scala version (2.10.x).

Running the Examples and Shell

这样安装的spark自带一些例子， Scala, Java, Python and R 的例子在 examples/src/main目录，可以在安装的spark根目录使用命令 bin/run-example [params] 跑一些例子，例如计算Pi

可以直接在控制台中敲入spark-shell，开始spark之旅了。
